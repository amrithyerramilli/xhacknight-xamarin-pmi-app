What is Match.AI ?

Match.AI is a REST API built over IBM Bluemix's Personality Insights Service.

What does Match.AI do ?

Match.AI uses a custom algorithm on top of Watson's PI service to generate a PMI Rating - Personality Match Index based on the user's personality. It uses Facebook to get social context of a user. IBM Bluemix's Personality Insights Service derives insights from transactional and social media data to identify psychological traits which determine purchase decisions, intent and behavioural traits.
